StockBase is a web-based inventory and order management system designed to allow users to manage items, categories, and customer orders through an intuitive interface. The application integrates a frontend built using HTML, CSS, and JavaScript with a backend database powered by Supabase The aim of the project is to demonstrate the development of a database-driven web application that supports full CRUD (Create, Read, Update, Delete) functionality while maintaining a clean and user-friendly design.
