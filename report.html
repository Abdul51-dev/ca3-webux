Project Report

System documentation and design overview

Overview

StockBase is a database-driven inventory and order management system. It allows users to manage items and orders using a web interface connected to Supabase.

Features

  • Manage items (Add, Edit, Delete)
  • Manage orders
  • Dashboard statistics
  • Real-time database updates

CRUD Functionality

OperationDescription
CreateAdd items/orders via forms
ReadDisplay data in tables
UpdateEdit existing data
DeleteRemove records

Database Tables

TableDescription
ItemsStores item data
OrdersStores order data
CategoriesStores categories
